## 支持粘贴 Markdown 内容

复制 Markdown 内容粘贴到文本编辑器中, 会自动进行转换.

粘贴的 Markdown 的语法遵循 VS Code 内置的 Markdwon 规范: 标准语法 + GFM 扩展语法 + Tatex 语法(`$文本$`和`$$文本$$`). 也就是说从 VS Code 中复制的 Markdown 内容可以直接粘贴到编辑器中, 唯一的例外是为了安全不支持 Markdown 中包含 html 标签.

## 拖拉支持

已插入的图片支持拖动到任意位置; 当鼠标移动到图片右下角会出现指示器, 拖动指示器可以调整图片大小.

## Mardown 语法识别

Mardown 语法识别遵循 Markdown 的规范

**Mardown 输入规则自动转换需要紧跟空格加以确认**, 并且可以在转换后按<kbd>Backspace</kbd>取消自动转换.

| 输入                          | 转换       | 说明                                                                                                                                     |
| ----------------------------- | ---------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| `#`+<kbd>空格</kbd>           | H1         | 必须在行头有效                                                                                                                           |
| `##`+<kbd>空格</kbd>          | H2         | 必须在行头有效                                                                                                                           |
| `###`+<kbd>空格</kbd>         | H3         | 必须在行头有效                                                                                                                           |
| `####`+<kbd>空格</kbd>        | H4         | 必须在行头有效                                                                                                                           |
| `#####`+<kbd>空格</kbd>       | H5         | 必须在行头有效                                                                                                                           |
| `######`+<kbd>空格</kbd>      | H6         | 必须在行头有效                                                                                                                           |
| `---`                         | 水平分割线 | 必须在行头有效                                                                                                                           |
| `-`或`+`或`*`+<kbd>空格</kbd> | 符号列表   | 必须在行头有效                                                                                                                           |
| `1.`+<kbd>空格</kbd>          | 序号列表   | 必须在行头有效, 任意数字均可                                                                                                             |
| `[ ]`或`[x]`+<kbd>空格</kbd>  | 任务列表   | 必须在行头有效, `[ ]`中间有个空格,表示未选中状态; x 字符必须小写,表示选中状态                                                            |
| \`\`\`                        | 围栏代码块 | 必须在行头有效, \`\`\`之后可以紧跟语言标识, 这会启用语法高亮. 支持的语言标识: txt,js,jsx,ts,tsx,html,css,c,cpp,md,java,csharp,sh,sql,ini |
| `$$`+<kbd>空格</kbd>          | 数学公式块 | 必须在行头有效                                                                                                                           |
| `[显示文本](链接)`            | 插入链接   | 可以在任意位置                                                                                                                           |
| `![显示文本](链接)`           | 插入图片   | 可以在任意位置                                                                                                                           |

**Mardown 包裹规则自动转换需要前导符之前至少有一个空格, 否则会识别为常规字符. 可以在行中任意位置**

| 语法                      | 转换                      | 说明                                          |
| ------------------------- | ------------------------- | --------------------------------------------- |
| \`文本\`                  | 行内代码                  | 可以在行中任意位置,但前导符之前至少有一个空格 |
| `_文本_` 或者`*文本*`     | 斜体文本                  | 可以在行中任意位置,但前导符之前至少有一个空格 |
| `**文本**` 或者`__文本__` | 粗体文本                  | 可以在行中任意位置,但前导符之前至少有一个空格 |
| `~~文本~~`                | 删除线文本                | 可以在行中任意位置,但前导符之前至少有一个空格 |
| `==文本==`                | 高亮文本                  | 可以在行中任意位置,但前导符之前至少有一个空格 |
| `$公式文本$`              | 内联的 Katex 数学公式     | 可以在任意位置                                |
| `$$公式文本$$`            | 块级居中的 Katex 数学公式 | 可以在任意位置                                |

## 其它操作

- 在表格中

  在**最右下角**的单元格中按下<kbd>Tab</kbd>键自动添加行.

- 在列表中

  在**末尾的空项**处, 每按一次<kbd>Enter</kbd>键会退出一级, 直至退出整个列表

### 间隙光标

正常的光标是闪烁的垂直线条, 间隙光标是闪烁的水平线条. 间隙光标不是真实的位置, 只有输入内容后才会在间隙光标处创建插入点.

间隙光标出现时可以直接输入, 编辑器会自动创建插入点. 间隙光标是为解决在文档最前面或者最后面处没有插入点的问题, 当遇到这个问题时通过以下方式显示间隙光标:

- 对于表格:
  在**最左上角**的单元格中按<kbd>↑</kbd>键会显示表格上方的间隙光标
  在**最右下角**的单元格中按<kbd>↓</kbd>键会显示表格下方的间隙光标

- 对于图片
  选定图片后按下<kbd>↑</kbd>键/<kbd>↓</kbd>键, 显示图片上方/下方的间隙光标
