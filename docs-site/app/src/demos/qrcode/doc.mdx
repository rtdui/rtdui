## 二维码纠错级别

`QRCode`支持四级纠错级别: L,M,Q,H, 分别对应: 低,中,高,超高. 默认为`M`

纠错级别越高生成图像越复杂, 但具有较高的纠错能力, 因此中心可被遮挡的部分也就越大.
