# 使用

`Avatar`组件用于展示单个用户头像

<Demo data={{ Component: AvatarBasicDemo }} />

## 变体

`Avatar`的`variant`属性支持: "square" 和"circle" 两种变体

## 尺寸

`Avatar`的`size`属性支持: "xs","sm","md","lg", 默认为"md"

## 是否在线指示器

`Avatar`的`online`属性显示是否在线指示器, 默认为`undefined`, 即不显示指示器.

## 字符占位

`Avatar`可以不显示头像, 只显示字符作为图像占位.

使用`placeholder`属性替代`src`属性即可.

> tips: 建议只使用单个字符作为占位.

<Demo data={{ Component: AvatarPlaceholderDemo }} />

## 遮罩

`Avatar`预定义了9种遮罩效果: "squircle"(方圆形), "heart"(心形), "star"(星形), "triangle"(三角形), "diamond"(钻石四边形), "pentagon"(五边形), "hexagon"(六边形), "hexagon2"(六边形2), "decagon"(十边形)

可以通过`mask`属性设置:

<Demo data={{ Component: AvatarMaskDemo }} />

## 头像组

`AvatarGroup`组件用于展示一组用户头像, 最多显示 4 个头像, 超出后会显示剩余的数量.

<Demo data={{ Component: AvatarGroupDemo }} />
