# 使用

Accordion 每次只能展开一项

### 默认,无展开/收缩指示器

<Demo
  data={{
    Component: AccordionNoIndicatorDemo,
  }}
/>

### 展开/收缩 箭头指示器

<Demo
  data={{
    Component: AccordionArrowIndicatorDemo,
  }}
/>

### 展开/收缩 加/减号指示器

<Demo
  data={{
    Component: AccordionPlusIndicatorDemo,
  }}
/>

### 设置初始展开项

<Demo
  data={{
    Component: AccordionInitExpandDemo,
  }}
/>

### 允许同时展开多项

默认同时只能展开一项, 设置`onlyOneOpen`属性为`false`, 允许同时展开多项

<Demo
  data={{
    Component: AccordionMultOpenDemo,
  }}
/>

### 拼合模式

默认项与项之间是分离的, 设置`join`属性为`true`, 则采用拼合模式, 会自动处理项的边框圆角

<Demo
  data={{
    Component: AccordionJoinItemsDemo,
  }}
/>
