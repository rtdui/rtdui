# 使用

`YearPickerInput`, `MonthPickerInput`, `DatePickerInput`, `DateTimePickerInput`, `DateInput` 组件支持清除, 设置它们的`clearable`属性为`true`, 当有输入内容时会在输入右侧带一个清除按钮.
