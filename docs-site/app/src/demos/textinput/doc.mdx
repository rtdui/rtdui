# DebouncedInput

`DebouncedInput`组件会延迟得到值, 对于根据用户输入执行后台查询这种场景尤其有用.

`DebouncedInput`的默认延迟等待时间是`500ms`, 延迟超时后才会触发`DebouncedInput`的`onChange`事件, 如果在等待期内用户再次输入则会取消等待, 输入后重新启动计时.
