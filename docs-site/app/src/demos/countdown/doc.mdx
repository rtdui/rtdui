# 使用

拥有动画效果的倒计时组件.

组件会根据传入的`endDate`属性自动计算应该显示哪些时间节点. 比如: 截止日期距今是 1 年以内的. 则只会从月开始显示, 同理,截止日期距今是 1 月以内的, 则只会从日开始显示, 以此类推. 最多显示到年.
