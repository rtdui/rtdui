# 使用

`Modal`组件呈现模态框, 模态框打开时会阻止与外部元素的交互, 直到关闭模态框.

<Demo data={{ Component: ModalBasicDemo }} />

## 移除头部

未指定`title`属性,并且`withCloseButton`为false, 则会移除头部

<Demo data={{ Component: ModalNoHeaderDemo }} />

`position`属性设置模态框的位置.

顶部(top):
