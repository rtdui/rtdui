# 使用

AutoComplete 支持键盘导航:

- 在 AutoComplete 得到焦点时, 下拉列表会立即显示(可以按键盘的上/下箭头键滚动下拉列表), 一旦用户输入会随着输入自动过滤列表项.

- 当下拉列表的某一项被键盘导航激活时, 按下键盘的回车键(`Enter`)则会自动将其作为输入值
