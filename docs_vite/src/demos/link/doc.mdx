## 多态

`Link`是一个[多态组件](/polymorphic), 其`as`属性是多态属性. 默认为HTML原生的`a`元素组件. 通过修改`as`属性, 可以很容易的与其它框架提供的组件进行集成. 如使用`react-router-dom`提供的`Link`组件替代`a`组件.
