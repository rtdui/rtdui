# 使用

## 基本

DataTable 列过滤支持类似于`Excel`过滤列表的功能. 要想启用功能,只要在列定义中的 meta 元属性中启用`showFilterList`即可:

```js
{
  // ...
  meta: {
    showFilterList: true,
  }
}
```

DataTable 会自动根据数据源中该列的唯一值生成过滤列表.
