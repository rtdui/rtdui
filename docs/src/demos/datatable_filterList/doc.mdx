# 使用

## 基本

DataTable 列过滤不仅支持常规的过滤, 还支持类似于`Excel`过滤列表的功能. DataTable 会自动根据数据源中该列的唯一值生成过滤列表.

要想启用列的过滤列表功能, 需要在列定义中的 meta 元属性中启用`showFilterList`:

```js
{
  // ...
  meta: {
    showFilterList: true,
  }
}
```
