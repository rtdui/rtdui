# RadioGroup 组件

`RadioGroup`组件为`Radio`提供了上下文, `RadioGroup`的`name`,`size`,`color`属性为`Radio`孩子提供公用属性值.

`RadioGroup`组件的`value`属性是受控属性, `defaultValue`是非受控属性, 当选中项改变时会触发`onChange`事件.
