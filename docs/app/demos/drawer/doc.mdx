# 使用

默认的, Drawer会充满视口. (CSS的postion为fixed)

<Demo data={{ Component: DrawerBasicDemo }} />

可以给`Drawer`指定`target`属性, Drawer会定位在`target`内(CSS的postion为absolute)

注意: `target`元素必须设置`relative`相对定位, 而且应该给它一个高度.
