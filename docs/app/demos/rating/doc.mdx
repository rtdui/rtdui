# 使用

## 基本

Rating 有五颗星, 默认 Rating 表示 5 分值.

> Tip: 可以点击第一颗心的左侧空白处清除选择.

<Demo data={{ Component: RatingBasicDemo }} />

## 启用半星级

如果`half`属性为 true, 则 Rating 表示 10 分值.

<Demo data={{ Component: RatingHalfDemo }} />

## 受控模式

`defaultValue`属性为非受控属性, `value`属性为非受控属性
