# 使用

Popover 基于 [floating-ui](https://floating-ui.com/) 实现. 可以完全自定义弹出的内容.

`Tooltip`,`Select`,`AutoComplete`组件等都是基于`Popover`实现

## 焦点围栏

焦点围栏是指当弹出框显示时, 键盘`Tab`键的焦点跳转范围限定在弹出框内. 对于弹出框中带有输入控件时尤其有用.
