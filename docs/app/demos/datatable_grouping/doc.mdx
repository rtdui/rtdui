# Grouping

DataTable 支持用户拖拉列头进行数据分组

设置`enableGrouping`属性为`true`开启用户数据分组功能. 可选的, 设置`autoExpandAll`为`true`分组后自动展开所有分组.

默认的, 分组后在分组单元格内自动显示组内行数.
