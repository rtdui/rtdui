# 使用

AspectRatio 使内容保持一致的宽/高比。 它可用于显示图像, 地图, 视频和其他媒体。

AspectRatio 根据元素的`width`的 css 计算值和提供的`radio`属性自动计算元素的`height`值, 即便元素的`width`为`auto`也可以.

AspectRatio 使用了 `CSS4` 规范中的`aspect-ratio`属性, 目前支持度达`91%`, `Safari`浏览器从`15`开始支持
