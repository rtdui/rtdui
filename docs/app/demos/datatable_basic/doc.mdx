# 使用

## 基本

DataTable 默认启用的功能: 固定列头行, 排序, 显示边框, 隐藏/显示列, 列宽调整(用户可以拖拉列头调整列宽), 列序调整(用户可以拖放调整列顺序),

排序支撑多列排序. 按住`Shift`键点击列头排序会按点击的先后顺序级联排序.
